## SDK v3 (recommended)

Bundled npm package, Shadow DOM modal, headless mode for backends. Pick the channel that fits — full step-by-step guides per channel are in the [SDK v3 Quickstart](/docs-v2/sdk-v3/quickstart).

<Steps>

### Step 1: Create a paywall

In the dashboard → **Paywalls** → **New Paywall**. Configure plans and payment processor.

### Step 2: Install the SDK

```bash
pnpm add @monetize.software/sdk
```

### Step 3: Open the paywall (web / SPA)

```ts
import { PaywallUI } from '@monetize.software/sdk/ui';

const paywall = new PaywallUI({
  paywallId: '3',
  apiOrigin: 'https://YOUR_DOMAIN',
  auth: true // built-in email / OTP / OAuth login
});

document.getElementById('upgrade')!
  .addEventListener('click', () => paywall.open());

paywall.on('purchase_completed', () => unlockPremium());
```
